## 📌 О проекте

Программа предназначена для автоматизированного расчёта **общей суммы подоходного налога** по данным компании. Все вводимые значения проходят строгую валидацию с проверкой диапазонов и формата данных.

Проект разработан в рамках учебного задания по программированию на C++.

---

## ✨ Возможности

- ✅ **Валидация ввода** — проверка на корректность, диапазон и пустые значения
- ✅ **Гибкие ограничения** — индивидуальные мин/макс для каждого поля
- ✅ **Точные вычисления** — использование типа `double` и фиксированной точности
- ✅ **Поддержка кириллицы** — корректное отображение русского текста в консоли Windows
- ✅ **Объектно-ориентированный подход** — инкапсуляция данных в классе `HRDepartment`
- ✅ **Защита от ошибок** — обработка исключений и проверка на переполнение
