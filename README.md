# Лабораторная работа по программированию

## Описание
Программа для расчета общей выплаты по подоходному налогу в департаменте HR.

## Функциональность
- Ввод данных о компании (название, количество сотрудников)
- Расчет нормы времени и почасовой оплаты
- Автоматический расчет подоходного налога
- Валидация вводимых данных

## Технические детали
- **Язык программирования:** C++
- **Стандарт:** C++11 и выше
- **Среда разработки:** Microsoft Visual Studio
- **Класс:** HRDepartment
- **Основные методы:**
  - `setCompanyName()` - установка названия компании
  - `setEmployeeCount()` - установка количества сотрудников
  - `setMonthlyProductionNorm()` - установка нормы времени
  - `setHourlyRate()` - установка почасовой ставки
  - `setIncomeTaxRate()` - установка процентной ставки налога
  - `calculateTotalIncomeTax()` - расчет общей суммы налога

## Как запустить в Visual Studio

### Вариант 1: Открыть существующий проект
1. Запустите **Visual Studio**
2. Нажмите **Файл** → **Открыть** → **Проект или решение**
3. Выберите файл `.sln` или `.vcxproj` (если есть)
4. Нажмите **F5** или кнопку ▶️ **Локальный отладчик** для запуска

### Вариант 2: Создать новый проект и добавить файл
1. В Visual Studio: **Файл** → **Создать** → **Проект**
2. Выберите шаблон: **Пустой проект (C++)** → **Далее**
3. Укажите имя проекта (например, `lab1`) и расположение → **Создать**
4. В **Обозревателе решений** кликните правой кнопкой по папке **Исходные файлы**
5. Выберите **Добавить** → **Существующий элемент**
6. Выберите файл `main.cpp` из этого репозитория
7. Нажмите **F5** для запуска программы

### Вариант 3: Просто скомпилировать файл
1. Создайте пустой проект C++ (как в Варианте 2)
2. Добавьте `main.cpp` в проект
3. Убедитесь, что в свойствах проекта установлен стандарт C++11 или выше:
   - **Проект** → **Свойства** → **C/C++** → **Язык** → **Стандарт языка C++** → `ISO C++11` или новее
4. Нажмите **Ctrl+F5** для запуска без отладки
